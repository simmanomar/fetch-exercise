# fetch-exercise

Basic fetch exercise, using PicSum API for the images (https://picsum.photos/).

https://simmanomar.github.io/fetch-exercise/

Learning how to fetch and display images from API. I learned how to randomly shuffle the images in the array so that each refresh of the browser would show a new image. I also got some practice in creating a scroll to top button and styling it; so that it is visible only when you scroll a certain distance from the top. I have added a load more button at the bottom but will continue to work on connecting it so that more images load when clicked further down the line.
